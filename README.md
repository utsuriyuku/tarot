# Quantum Tarot (量子塔罗)

## 🌌 项目理念

本项目旨在突破传统占卜类应用的窠臼。在这里，交互是用户的眼睛，页面是美丽的表皮，而内核则是对时间与因果的重新解构。

**核心世界观**：时间本身是不存在的。过去、现在和未来是在同一维度的全息幻象。当你抽出卡牌的那一刻，并非“过去导致了现在的抽牌”，而是“特定的未来在这一刻坍缩，决定了你抽中这些牌”。

在这个框架下，塔罗牌不仅是中世纪的神秘学工具，也是一种建立在“量子坍缩”和“非线性时间观”上的心理映射。

## 🛠️ 技术栈
- **核心框架**: React + Vite (或 Next.js)
- **3D/粒子交互**: Three.js + React Three Fiber (@react-three/fiber)
- **页面样式**: Tailwind CSS
- **大模型解析 API**: 结合自定义的系统级 Prompt，连接 LLM 提供千人千面的超凡洞见。
